MP3 Gallery Script:

Description: MP3 Gallery Script is music streaming and downloading script. The Script allows the site owners can create different albums under different categories and upload MP3 files.

MP3 Gallery Script was developed using Laravel and Bootstrap which allows administrator to escalate the application as his/her desire using this frameworks.

Features:
  • Create Albums
  • Upload MP3 files
  • Add Album cover to MP3 files
  • Download and Stream Music
  • Download full Albums (ZIP Download)
  • Fully Ajax Based
  • Users can create Playlists
  • Email verification
  • Social Network Integration (Facebook, Twitter, Google)
  • Poweful Admin dashboard
  • 100% Responsive Design
  • And many more...

Script Requires:

  • Operating System : Linux
  • Web Server : Apache
  • PHP Version: 5.4+
  • PHP Allow_URL_Fopen ON
  • PHP FileInfo Module Enabled
  • PDO extension
  • PHP GD Extension
  • PHP CURL Enabled
  • Mod_Rewrite Module Enabled
  • Mysql 5.x
